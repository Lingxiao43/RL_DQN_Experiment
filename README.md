# Deep RL GridWorld: 从表格型 Q-Learning 到深度 DQN 的跨越

本项目探索了如何在网格迷宫（GridWorld）环境中，将传统的表格型强化学习（Tabular Q-Learning / Monte Carlo, C++实现）成功迁移并重构为基于深度神经网络的 **Deep Q-Network (DQN)**（Python + PyTorch 实现）。

在此过程中，我们经历了从“网络发散崩溃”到“完美收敛”的完整试错过程，并通过严格的**控制变量法（Ablation Study）**，推导出了深度强化学习中关于**状态表征、信噪比（SNR）、过估计偏差**的底层硬核逻辑。

## 💡 核心实验发现与理论重构 (Key Takeaways)

在将算法从 C++ 迁移到 PyTorch 神经网络的过程中，我们解决了深度强化学习中几个极其经典的“大坑”：

### 1. 状态表征的降维打击：为什么连续坐标会失败？
*   **陷阱**：直接将坐标 $(x, y)$ 作为神经网络的输入会导致失败。因为迷宫存在非线性的“断层”（差一个格子，奖励从 $+100$ 突变到 $-10$），而全连接层（`Linear`）天生具有平滑泛化属性，无法拟合这种尖锐的悬崖。
*   **解药**：使用 **`nn.Embedding(grid_size * grid_size, 128)`**。这相当于赋予了神经网络“伪装成 C++ 独立表格”的能力，从底层切断了相邻状态之间的灾难性干扰（Catastrophic Interference）。

### 2. 拒绝 $\gamma=1.0$：信噪比（SNR）与绝对数值差的觉醒
*   在表格型 RL 中，$\gamma=1.0$ 配合走步微小惩罚是可行的（因为差 0.001 也能被数组精确区分）。
*   但在 DQN 中，$\gamma=1.0$ 或过高的 $\gamma$（如 0.99）会导致远处相邻格子的目标价值差异（梯度信号）被神经网络的 SGD 随机噪音吞噬。
*   **最小有效信号量公式**：$\Delta Q = R_{goal} \times \gamma^{d-1} \times (1 - \gamma)$。
*   **结论**：在深度 RL 中，不仅“变化率”重要，“绝对数值差距”同样致命。通过**调低 $\gamma$（如 0.96~0.98）**并**大幅放大终点奖励（如 100 $\to$ 1000）**，我们强行拉大了相邻格子的绝对差值，使其击穿了网络的初始底噪，网络瞬间恢复收敛。

### 3. “原地发呆 (STAY)” 动作的自举死循环
*   实验发现，在 $\gamma$ 较高且存在 `STAY` 动作时，智能体会陷入无尽的“发呆”或绕圈子。
*   **数学本质**：DQN 的 $\max$ 操作符带有强烈的**过估计偏差（Overestimation Bias）**。一旦噪音导致 $Q(s, STAY)$ 略高于最优动作，网络就会使用 $Q(s, STAY) = R_{step} + \gamma Q(s, STAY)$ 进行自我更新（左脚踩右脚），形成极其顽固的虚假高分海市蜃楼。去除 `STAY` 动作（`NUM_ACTIONS=4`）是打破这一魔咒的有效物理手段。

## 🛠️ 环境配置与安装

本项目深度依赖 GPU 加速计算（特别是大地图的经验回放与网络更新）。建议使用 NVIDIA GPU 运行。

**1. 创建并激活虚拟环境 (推荐):**
```bash
python -m venv .venv
# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate
```

**2. 安装 PyTorch (带 CUDA 支持，以 RTX 40/30 系列为例):**
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
```

**3. 安装其他依赖:**
```bash
pip install numpy
```
